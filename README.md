## 🚀 Overview

**Essem Connect** is a centralized web application designed to streamline wire harness manufacturing operations. From live production planning to dynamic Bill of Materials (BOM) viewing, this platform empowers floor supervisors to manage inventory, track rejections, and communicate issues in real-time.

---

## ✨ Key Features

### 📋 Smart Floor Logs
* **Daily Production Planning:** Real-time logging of daily targets and achievements.
* **Delivery Challan (DC):** Track planned vs. dispatched quantities seamlessly.
* **OT Details:** Log operator overtime, worker names, and specific tasks completed.
* **4M Forms:** Quickly record operational changes across **M**an, **M**achine, **M**aterial, and **M**ethod.

### 📦 Inventory & Model Management
* **Model Directory:** Searchable database of all harness models by FG Number, complete with harness reference images and a dynamic BOM viewer.
* **Material Inventory:** Auto-generated, deduplicated list of all raw materials extracted directly from the master BOM sheets.
* **Material Need Tracker:** Log required raw materials and shortages directly to a specific production line.

### 📊 Quality & Support
* **Advanced Defect Tracking:** Specialized, isolated logging interfaces for Assembly and Crimping rejections.
* **Automated Ticketing:** Integrated real-time feedback and IT/Maintenance support system (Powered by Google Apps Script).
